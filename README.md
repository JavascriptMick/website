![](https://raw.githubusercontent.com/diasdavid/specs/libp2p-spec/protocol/network/figs/logo.png)

# Development

Make sure to have a ipfs daemon running and leverage nodemon for the win!

```
$ npm run mon
# Then open localhost:9090
```
